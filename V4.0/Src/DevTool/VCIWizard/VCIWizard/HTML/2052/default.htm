Script.NET组件向导说明

 

此向导可以生成符合Script.NET平台要求的组件工程。

 

Script.NET组件有多种类型，您可以选择生成哪种类型的组件，建议在Script.NET的Pligins目录下先建立相应组件的子目录，然后将工程生成到此目录中，并修改生成的dll的目录到组件目录下，组件工程需要依赖于Script.NET\Plugins\Include目录下的头文件，如果按照以上建议的路径，则默认生成的工程已经添加了头文件路径，直接编译就可以。组件在Script.NET平台运行时候还需要有plugin_cn.xml配置文件，请参考平台中其他组件的配置文件来建立组件的配置文件。
